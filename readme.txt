This widgets returns a comprehensive list of local bicycle rides and races in your area based on the users location or a US ZIP code and search radius.

== Description ==

Include a comprehensive listing of upcoming bicycle rides, races and events on your sidebar with links to more information.  
Pick a U.S. ZIP code, and the widget finds nearby events within a search radius.  
Pick the number of events to display.

Currently, this only works for the United States with U.S. ZIP codes.  We are working on a version that will include Canada, England and Australia.
If you know of a good source for cycling events in those countries, please post a comment at http://www.freehub.net/bike-events-widget
